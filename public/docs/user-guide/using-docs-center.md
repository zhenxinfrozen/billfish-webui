# 文档中心使用指南

## 什么是文档中心?

文档中心是Billfish Web Manager 的知识库,包含系统的所有使用文档、开发指南和故障排除方案。

## 📚 文档分类

### 🚀 入门指南
适合新用户快速了解系统功能和使用方法。

**推荐阅读顺序:**
1. 快速入门- 5分钟了解核心功能
2. v0.1.0 快速开始- 版本特定的安装配置

### 📖 用户指南
详细的功能使用说明和最佳实践。

**内容包括:**
- 文件浏览和管理
- 搜索和过滤技巧
- 标签使用方法
- 预览图查看

### 🔧 开发文档
面向开发者的技术文档和API参考。

**内容包括:**
- 数据库映射系统
- BillfishManagerV2 API
- 文档和工具系统架构
- 扩展开发指南

### 📋 版本说明
系统的版本更新日志和功能变更记录。

**内容包括:**
- v0.1.0 发布说明
- 版本总结
- 升级指南

### 🔍 故障排除
常见问题解决方案和调试技巧。

**内容包括:**
- 预览图缺失问题
- 数据库连接错误
- 权限问题
- 性能优化

## 🧭 导航方式

### 侧边栏导航
- 左侧显示所有文档分类
- 点击分类展开文档列表
- 点击文档标题打开内容
- 当前文档高亮显示

### 面包屑导航
- 页面顶部显示当前位置
- 格式: 文档中心 > 分类名称 > 文档标题
- 点击面包屑可快速返回上一级

### 首页概览
- 显示所有分类的卡片
- 显示每个分类的文档数量
- 点击卡片进入分类

## 🔍 搜索功能

### 全文搜索
1. 点击搜索框
2. 输入关键词
3. 实时显示搜索结果
4. 高亮显示匹配内容

### 搜索技巧
- 使用空格分隔多个关键词
- 关键词不区分大小写
- 支持中文和英文搜索
- 搜索文档标题和内容

## 📱 响应式设计

### 桌面端
- 左侧固定导航栏(280px)

### 移动端
- 导航栏自动折叠
- 垂直堆叠布局
- 触摸友好的交互

## 💡 使用技巧

### 快速查找
1. **按分类浏览** - 知道文档类型时最快
2. **使用搜索** - 查找特定关键词
3. **查看首页** - 了解所有可用文档

### 收藏常用文档
- 浏览器书签保存文档链接
- URL格式: `docs-ui.php?section=分类ID&file=文件名.md`
- 示例: `docs-ui.php?section=getting-started&file=quick-start.md`

### 与工具中心联动
- 文档中提到的工具可在工具中心找到
- 链接格式: `[工具名称](../tools-ui.php?tool=工具ID)`
- 示例: [数据库分析工具](../../tools-ui.php?tool=deep-analysis)
