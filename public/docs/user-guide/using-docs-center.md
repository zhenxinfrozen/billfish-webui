# 文档中心使用指南

欢迎使用 Billfish Web Manager 文档中心！本指南将帮助您快速了解如何使用文档系统。

## 文档中心功能

### 1. 文档浏览

- **分类导航**：左侧边栏按主题分类
- **快速搜索**：顶部搜索栏快速查找
- **面包屑**：显示当前位置
- **目录跳转**：点击标题快速定位

## 使用技巧

### 侧边栏导航

1. **展开/折叠分类**：点击分类标题
2. **高亮当前页**：当前文档以蓝色高亮
3. **记忆状态**：浏览器记住展开/折叠状态

### 文档内导航

1. **标题锚点**：点击标题复制链接
2. **代码高亮**：自动识别代码语言
3. **链接跳转**：支持内部和外部链接

### 搜索功能

1. 使用搜索框输入关键词
2. 系统搜索文档标题和内容
3. 点击结果快速跳转

## 文档编辑

### 文档格式

所有文档使用 Markdown 格式编写，存放在 `public/docs/` 目录。

### 目录结构

```
docs/
├── getting-started/    # 入门指南
├── user-guide/         # 用户指南
├── development/        # 开发文档
├── setup/              # 安装配置
├── release-notes/      # 版本说明
├── troubleshooting/    # 故障排除
└── config.json         # 文档配置
```

### 添加新文档

1. 在对应目录创建 `.md` 文件
2. 编辑 `config.json` 添加文档信息
3. 刷新页面查看效果

### 配置示例

```json
{
  "sections": [
    {
      "id": "getting-started",
      "name": "入门指南",
      "icon": "🚀",
      "order": 1,
      "documents": [
        {
          "file": "quick-start.md",
          "title": "快速开始",
          "description": "快速上手指南",
          "order": 1,
          "badge": "推荐"
        }
      ]
    }
  ]
}
```

## 文档规范

### Markdown 语法

支持标准 Markdown 和扩展语法：

#### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

#### 代码块
```markdown
​```php
<?php
echo "Hello World";
?>
​```
```

#### 列表
```markdown
- 无序列表项1
- 无序列表项2

1. 有序列表项1
2. 有序列表项2
```

#### 链接
```markdown
[文字链接](https://example.com)
[内部链接](../setup/config.md)
```

#### 图片
```markdown
![替代文字](image.png)
```

## 键盘快捷键

- `Ctrl + F`：页内搜索
- `Ctrl + K`：聚焦搜索框
- `Esc`：关闭搜索/对话框

## 移动端使用

- 点击汉堡菜单展开侧边栏
- 滑动浏览文档
- 双击标题返回顶部
