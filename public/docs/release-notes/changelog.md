# 更新日志

记录 Billfish Web Manager 的版本更新历史。

## v0.0.5 (2025-01-17)

### 🎨 UI 重构
- 文档中心完全重新设计，采用 Wiki.js 风格
- 优化侧边栏导航，增强视觉层级
- 修复标签换行和溢出问题
- 改进配色方案，提升可读性

### 🐛 Bug 修复
- 修复中文编码乱码问题
- 移除不必要的 UTF-8 BOM
- 修复CSS样式被打印到页面底部的问题

### 📚 文档
- 重新生成所有核心文档
- 添加详细的配置指南
- 完善故障排除文档
